Instalar Angular

```
npm i -g @angular/cli@14.2.12 --force
```

Clonar el repositorio

```
git clone https://github.com/valentinagn13/client-interface.git
```

Acceder a la carpeta "frontend"

```
cd frontend
```

Instalar las dependencias

```
npm install --force
```

Correr el servidor

```
ng serve
```
